Grid CSS

El Grid en el formato de estilo CSS facilita el diseño y la distribución de elementos a lo largo de una pagina Web. El sistema grid define una cuadrícula determinada, y modificando los nombres de las clases de los elementos HTML, podemos darle tamaño, posición o colocación.


Grid CSS nace de esa necesidad, y recoge las ventajas de ese sistema, añadiendole numerosas mejoras y características que permiten crear rápidamente cuadrículas sencillas y potentes de forma prácticamente instantánea.


Los conceptos usados en el sistema Grid constan:



    Contenedor: El elemento padre contenedor que definirá la cuadrícula o rejilla.

    Ítem: Cada uno de los hijos que contiene la cuadrícula (elemento contenedor).

    Celda (grid cell): Unidad mínima de la cuadrícula.

    Área (grid area): Región o conjunto de celdas de la cuadrícula.

    Banda (grid track): Banda horizontal o vertical de celdas de la cuadrícula.

    Línea (grid line): Separador horizontal o vertical de las celdas de la cuadrícula.

Una cuadricula se usa y muestra conteniendola en la propiedad Display y se valora de la siguiente manera para influir en su comportamiento en el contenido exterior:

    inline-grid: Establece una cuadrícula con ítems en línea, de forma equivalente a inline-block. Permite que la cuadrícula aparezca encima/debajo del contenido exterior

    grid:Establece una cuadrícula con ítems en bloque, de forma equivalente a block. Permite que la cuadrícula aparezca a la izquierda/derecha (en línea) del contenido exterior

La cuadricula puede contener determinadas filas y columnas, se definen con la propiedad grid-template-columns y grid-template-rows, que dimensiona cada celda. En el codigo se define de la forma:


                    .grid {
                        display: grid;
                        grid-template-columns: 50px 300px; /* definicion de 2 columnas */
                        grid-template-rows: 200px 75px;  /* definicion de 2 filas */
                    }

El valor del ancho se puede determinar tambien por unidad de fracción restante que quiere decir fracción de espacio restante en el grid(fr), por porcentaje o en auto.


Definición de áreas

En HTML se define las etiquetas sistemicas como por ejemplo:


                    div class="grid"
                    div class="header">Header /div
                    div class="sidebar">Sidebar /div
                    div class="content">Content /div
                    div class="footer">Footer /div
                    /div

En CSS se define una estructura definida, se define una estructura de lineas para el grid (ejemplo x para columnas; y para filas), tambien se puede usar cualquier nombre identificador. En el codigo CSS se definen los nombres entre corchetes


                .grid {
                    display: grid;
                    grid-template-columns: [x0] 1fr [x1] 1fr [x2] 1fr [x3];
                    grid-template-rows: [y0] 1fr [y1] 1fr [y2] 1fr [y3];
                }

Por ultimo se define cada etiqueta sistematica en el grupo de bloques que se quieran ocupar. Para ello utilizaremos las propiedades grid-column-start, grid-column-end y grid-row-start, grid-row-end. Un ejemplo de este codigo seria:


                .header {
                    background: darkred;
                    grid-column-start: x0;
                    grid-column-end: x3;
              
                .content {
                    background: orange;
                    grid-column: x1 / x3;
                    grid-row: y1 / y3;
