Propósito del sitio


Con la nueva versión de HTML5 se añaden etiquetas para añadir valor semántico y estructural a la programación de una página Web.

Etiquetas semánticas


Las distintas partes de una página en su forma de diseño se distribuye en una cabecera, pie de página, navegación. etc. HTML5 facilita su diagramación de distribución con el uso de etiquetas para cada sección.

CSS Grid


El CSS Grid se usa en función del diseño, para lograr muchos diseños que el cliente y el programador puede realizar en el desarrollo de la página Web, permitiendo alinear elementos en columnas y filas, sin usar tablas. Es muy útil para los elementos secundarios de un contenedor de cuadrícula en donde se podrán posicionar para que se solapen y se superpongan.
