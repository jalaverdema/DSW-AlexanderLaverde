Etiquetas semánticas

Las etiquetas semanticas nacen debido a las divisiones estandar de las paginas web que siempre se repiten. En HTML5 se añaden las divisiones con significado para estructurar con facilidad. En la versión anterior de HTML el programador realizaba las divisiones con la etiqueta div etiquetando las partes de la pagina con un ID.

Las etiquetas semanticas mas importantes son:


    header: La cabecera de la pagina o seccion, puede haber mas de una en la pagina.

    footer: El pie de pagina donde tiene datos como el nombre del autor, copyright, terminos de uso, contacto entre otros.

    nav: Se definen los enlaces de navegacion del sitio web.

    section: Se define una seccion del documento. Representa a una sección genérica de una pagina. Agrupa contenidos que tienen una relación temática entre si.

    article: Se incluye el contenido independiente que se distribuye de forma lubre en el documento, ejemplo: un foro, noticias, comentarios, blogs, etc.

    aside: Sección de una pagina que contiene contenido relacionado tangencialmente al de su alrededor. Por lo general se utilizan como barras laterales.

    figure: Define un elemento independiente como una imagen, fragmento de codigo, entre otros. Este elemento debe estar relacionado con el elemento que lo contiene, o el de mas alto nivel. pero cuya posicion es independiente del mismo.

    figcaption: Es donde se contiene el titulo o leyenda del elemento y se coloca como primer y ultimo elemento.

    details: Son los detalles adicionales de la pagina que el programador puede ocultar.
